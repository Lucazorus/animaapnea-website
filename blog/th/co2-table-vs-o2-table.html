ตาราง CO2 vs ตาราง O2: ต่างกันอย่างไร?

12 เมษายน 2026 · อ่าน 7 นาที

ในโลกของการดำน้ำแบบ freediving คำถามที่ได้ยินบ่อยที่สุดคือ "ควรฝึกตาราง CO2 หรือ O2 ก่อน?" คำตอบขึ้นอยู่กับว่าคุณต้องการพัฒนาอะไร เพราะตารางทั้งสองประเภทนี้ทำงานต่างกันอย่างสิ้นเชิงและพัฒนาทักษะที่ต่างกัน

พื้นฐานทางวิทยาศาสตร์

เพื่อให้เข้าใจความแตกต่างได้อย่างชัดเจน ต้องเข้าใจก่อนว่าร่างกายทำงานอย่างไรระหว่างการกลั้นหายใจ

เมื่อเราหยุดหายใจ ร่างกายยังคงใช้ออกซิเจน (O2) และผลิตคาร์บอนไดออกไซด์ (CO2) อย่างต่อเนื่อง ระดับ O2 จะค่อยๆ ลดลง ในขณะที่ระดับ CO2 จะสูงขึ้น ความรู้สึกอยากหายใจเกิดขึ้นเมื่อระดับ CO2 สูงเกินเกณฑ์ที่ร่างกายยอมรับ ไม่ใช่เมื่อ O2 หมด นี่คือสิ่งสำคัญที่หลายคนเข้าใจผิด

ตาราง CO2: ฝึกความทนทานต่อ CO2

ตาราง CO2 ออกแบบมาเพื่อฝึกให้ระบบประสาทส่วนกลางและ chemoreceptors (ตัวรับสารเคมีในเลือด) คุ้นชินกับระดับ CO2 ที่สูงขึ้น เมื่อคุณฝึกซ้ำๆ ร่างกายจะปรับ threshold ให้สูงขึ้น ทำให้ความรู้สึกอยากหายใจเกิดขึ้นช้าลง

ตาราง O2: ฝึกความจุออกซิเจน

ตาราง O2 ทำงานต่างกันอย่างสิ้นเชิง แทนที่จะฝึกให้ทนต่อ CO2 มันฝึกให้ร่างกายทำงานได้อย่างมีประสิทธิภาพมากขึ้นในสภาวะที่ O2 ต่ำ รวมถึงเพิ่มความจุออกซิเจนโดยรวม

ความแตกต่างหลักของทั้งสองตาราง

CO2 Table

  • ฝึกความทนทานต่อความรู้สึกอยากหายใจ
  • พัฒนาความแข็งแกร่งทางจิตใจ
  • เหมาะสำหรับผู้ที่หยุดกลั้นหายใจเพราะรู้สึกอยากหายใจมาก
  • ปลอดภัยกว่าสำหรับผู้เริ่มต้น
  • ผลที่เห็นได้เร็วกว่า (ภายใน 2-4 สัปดาห์)

O2 Table

  • ฝึกประสิทธิภาพการใช้ O2
  • พัฒนาความจุออกซิเจนในเลือด
  • เหมาะสำหรับนักดำน้ำที่ต้องการทำลายสถิติส่วนตัว
  • มีความเสี่ยงสูงกว่าถ้าทำผิดวิธี
  • ต้องการพื้นฐาน CO2 tolerance ที่ดีก่อน

ควรเริ่มจากอันไหน?

สำหรับผู้เริ่มต้นและระดับกลาง แนะนำให้เริ่มจากตาราง CO2 ก่อน เหตุผลคือ:

  1. ปลอดภัยกว่า: ตาราง O2 ที่ทำผิดวิธีอาจทำให้ O2 ลดต่ำมากจนเป็นอันตราย ในขณะที่ตาราง CO2 มีความเสี่ยงน้อยกว่า
  2. ผลเร็วกว่า: ความทนทานต่อ CO2 พัฒนาได้เร็วกว่าความจุ O2 ทำให้เห็นความก้าวหน้าได้ชัดเจนกว่า
  3. เป็นพื้นฐาน: ก่อนจะฝึก O2 table ได้อย่างมีประสิทธิภาพ คุณต้องมี CO2 tolerance ที่ดีพอก่อน

การผสมผสานทั้งสองตาราง

นักดำน้ำที่มีประสบการณ์มักฝึกทั้งสองตารางสลับกัน ตัวอย่างโปรแกรมในหนึ่งสัปดาห์:

  • วันจันทร์: CO2 table
  • วันพุธ: O2 table
  • วันศุกร์: CO2 table หรือ breathwork

การสลับกันแบบนี้ช่วยให้ร่างกายพัฒนาทั้งสองด้านพร้อมกัน และป้องกันการเกิด plateau ในการฝึก

สัญญาณว่าคุณต้องการ CO2 Table มากกว่า O2 Table

ถ้าคุณมักหยุดกลั้นหายใจ "เพราะทนไม่ได้" แต่ยังรู้สึกว่ามีออกซิเจนเหลืออยู่ นั่นแสดงว่า CO2 tolerance ของคุณต่ำ และตาราง CO2 จะช่วยได้มาก ในทางตรงข้าม ถ้าคุณรู้สึกว่าหยุดเพราะร่างกายอ่อนแรงหรือ O2 หมดจริงๆ ตาราง O2 อาจเหมาะสมกว่า
